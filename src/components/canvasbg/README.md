> canvas背景

## API

* props

1. w - 可选 Number类型 canvas背景的宽 默认为屏宽
2. h - 可选 Number类型 canvas背景的高 默认为屏高
3. ballnum - 必选 Number类型
4. r - 必选 Number类型

* events

无

* methods

无

## Example

\<canvasbg w="1000" h="500" r="40" ballnum="60"\>tool\</canvasbg\>
