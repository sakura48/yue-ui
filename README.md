# yue-ui

> 基于Vue的UI组件库

## 安装

```
# 安装依赖
npm install

# 启动热重载服务 localhost:8080
npm run dev

# 打包生产环境组件库
npm run build

# 打包单一组件
npm run build 'component name'

# build for production and view the bundle analyzer report
npm run build --report

# 静态代码检查
npm run unit

# 运行测试用例
npm test

# 启动demo页面服务
npm run web
```
